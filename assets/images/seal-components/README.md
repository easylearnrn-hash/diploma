# ACNHS Official Seal Components

This directory contains the individual components of the Armenian College of Nursing & Health Sciences official seal.

## Complete Seals
- `acnhs-seal.svg` - Full detailed seal with all components
- `acnhs-seal-simple.svg` - Simplified professional version

## Component Files (Layered)

### Base Structure
1. **01_outer_gold_ring.svg** - Outer decorative gold border
2. **02_inner_navy_circle.svg** - Main navy blue background circle

### Decorative Elements
3. **03_left_laurel.svg** - Left laurel branch (academic achievement symbol)
4. **04_right_laurel.svg** - Right laurel branch (mirrors left)
5. **06_stars_left.svg** - Left star cluster (excellence)
6. **07_stars_right.svg** - Right star cluster (mirrors left)

### Central Symbol
5. **05_rod_of_asclepius.svg** - Medical/healthcare symbol (staff with snake)

### Text Elements
8. **08_text_outer_ring.svg** - Curved text around perimeter
9. **09_text_center_ACNHS.svg** - Center "ACNHS" abbreviation
10. **10_text_location.svg** - "Yerevan, Armenia" location text
11. **11_text_est_2025.svg** - "EST. 2025" establishment year

### Effects
12. **12_gold_effects_overlay.png** - Metallic shine and texture overlay

## Color Palette
- **Primary Gold**: #eab308 (with gradient from #f4d03f to #c2910a)
- **Navy Blue**: #0a2540 to #020617
- **Accent Teal**: #2dd4bf
- **White**: #ffffff

## Usage
Import the complete seal or layer individual components in your preferred design software (Adobe Illustrator, Figma, etc.)
